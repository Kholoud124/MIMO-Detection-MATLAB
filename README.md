# MIMO Detection in Rayleigh Fading (MATLAB)

This project presents a BER performance comparison of various MIMO detection techniques over Rayleigh fading channels.

## Implemented Detectors
- Zero Forcing (ZF)
- MMSE
- ZF-SIC
- MMSE-SIC
- Maximum Likelihood (ML)

## System Model
- Modulation: QPSK
- Channel: Rayleigh fading
- MIMO Configurations:
  - BER vs Number of Receive Antennas (SNR = 10 dB)
  - BER vs SNR for 3Ã—3 MIMO

## How to Run
1. Open MATLAB
2. Run `MIMO_MMSE.m`

## Output
- BER vs Nr figure
- BER vs SNR figure
